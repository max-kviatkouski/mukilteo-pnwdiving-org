Mukilteo Current Measurements Project

Hello fellow divers!
This is an attempt to build a cheap DYI current meter and establish more or less regular mesurements of current at Mukilteo T-Dock.
This site just got created and will be updated soon with following:
  • Visualization of latest current measurements
  • Some comment/feedback section (I'm thinking about some crowd-sourcing calibration for the meter)
Meanwhile, should you be interested, you're welcome to visit project page at hackerday.io: https://hackaday.io/project/158083-cheap-underwater-tilt-current-sensor
There is also a precise (hopefully) map of Mukilteo coming up soon which I'm working on too. I may post work in progress here as well so you can tell me if I've missed any features.
